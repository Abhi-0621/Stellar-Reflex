# Stellar Reflex

Stellar Reflex is a web-based reaction-time game that measures how quickly a player responds to a visual signal.

The game waits for a random amount of time, turns the game box green, and then records how fast the player clicks. It calculates reaction time in milliseconds, detects early clicks, saves the best score, and gives an animal-inspired reflex ranking.

## Features

- Random wait timer
- Reaction time calculation in milliseconds
- Early-click detection
- Best score display
- Animal-inspired reflex rankings
- Space-themed design

## Built With

- HTML
- CSS
- JavaScript
